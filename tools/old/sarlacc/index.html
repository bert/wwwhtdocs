    • What is Project Sarlacc?

      • Project Sarlacc is an OrCAD to gEDA format schematic converter. This project is beta software, so it still has limitations. To get a better idea of what those are, read the code, and search for "BAD" in the comments. The homepage of Project Sarlacc can be found here. The homepage has more info on how to compile and run the converter.

    • What are the latest features in the current release?

      • Version 0.92

      Now properly converts part attributes, UREFs, devices, etc. Better handling of ground and power connections. Produces files which can be netlisted now!

    • What does it look like?

      • Sarlacc is a commandline utility, no screenshots.
