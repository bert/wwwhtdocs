Notes: (Be sure to read all the notes on this page!)

  • You can only post using a subscribed e-mail address. If you do not subscribe and attempt to post to one of the above lists, your post will never make it to the lists.

  • If you are having posting problems, make sure the address that you send e-mail is the same address you subscribed with.

  • Remember, the posting address is for list traffic only!

  • To send email to the list use the posting address.

  • Do NOT send subscription/unsubscription requests to the posting address.

  • All non-free commercial EDA advertisements (Spam) are NOT welcome on any of the above mailing lists.

  • Be sure to read and follow the guidelines presented on the following link: How To Ask Questions The Smart Way.

  • Do NOT send spam to or harvest e-mail address from any page on this site.

  • Do NOT send messages to any gEDA list in HTML. Plain text only please!

  • Sending small binary attachments to any of the the gEDA lists is okay as long as they are relevant to gEDA.

  • Click on the "Monthly" archive links to get a little more info about each list.

  • The "Monthly" archive is always up-to-date, while the "Chunks" tend to be updated only once in a while.

  • Look at the table above, if the Subscribe column for the list you want to subscribe to is "Web" then click on that link to subscribe. The rest of this webpage relates to the older lists which use an admin list address.

  • To subscribe send an email to the admin list address with the following in the body: 
       subscribe LIST-NAME

  • To unsubscribe send an email to the admin list address with the following in the body: 
       unsubscribe LIST-NAME

  • LIST-NAME is the name of the list (either geda-user or geda-pt)

  • For all old style list admin commands use the admin list address.
